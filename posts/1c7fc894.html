Obsidian Plugin - Tasks

发表于2024-05-30|更新于2024-06-09|ObsidianPlugin
|字数总计:678|阅读时长:2分钟|阅读量:

https://github.com/obsidian-tasks-group/obsidian-tasks/blob/main/README.md の 翻译。

简介

Tasks 是 Obsidian 知识库的任务管理。

用户指南 - 贡献指南 - 路线图

跟踪整个保管库中的任务。查询它们并在您想要的任何位置将它们标记为已完成。支持到期日、重复任务（重复）、完成日期、清单项子集和过滤。

您可以在任何视图或查询中切换任务状态，它将更新源文件。

对于每个版本的更改，请查看发布页面：https://github.com/obsidian-tasks-group/obsidian-tasks/releases 。

截图

  • 所有截图均假定 全局过滤器 #task 默认情况下未设置（另请参阅 安装）。
  • 主题是默认的 Obsidian 主题。

笔记 ACME 有一些任务。

笔记 Important Project 中也有一些任务。

笔记 Tasks 收集了保险库中的所有任务并使用查询显示它们。

Tasks: Create or edit 命令可帮助您编辑任务。

安装

按照以下步骤安装任务。

  1. 在 Obsidian 的社区插件浏览器中搜索 Tasks ；
  2. 在 Obsidian 设置中启用该插件（在 Community plugins 下找到 Tasks ）；
  3. 检查设置。尽早设置全局过滤器是有意义的（如果您需要的话）。
  4. 将 Toggle checkbox status 热键替换为 Tasks: Toggle Done 。
    • 我建议您删除原来的切换热键，并将 Tasks 切换设置为 Ctrl + Enter 。

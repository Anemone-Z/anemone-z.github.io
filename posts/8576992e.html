Obsidian Plugin - Projects

发表于2024-05-30|更新于2024-06-09|ObsidianPlugin
|字数总计:723|阅读时长:2分钟|阅读量:

https://github.com/marcusolsson/obsidian-projects/blob/main/README.md の 翻译。

简介

Projects 是 Obsidian 的一个插件，可让你管理和可视化项目管理的注释。

  • 从文件夹和数据视图查询创建项目。
  • 在四种不同的视图之间切换：表格、看板、日历和画廊。
  • 为每个项目配置注释模板。

例如，如果您是内容管理员，Projects 可以帮助您管理内容日历。创建草稿、跟踪其状态以及计划发布的时间。

安装

您必须关闭 限制模式 才能使用项目。

  1. 在 Obsidian 中，打开 设置 。
  2. 在 社区插件 下，选择 浏览 。
  3. 搜索 Marcus Olsson 的“项目”，然后选择它。
  4. 选择 安装 。

要开始使用项目，请按 Ctrl+P（或在 macOS 上按 Cmd+P）打开 命令面板，然后选择 项目：显示项目。

设计理念

在开发任何软件时，您经常会面临艰难的选择。为了帮助指导设计决策，本项目遵循以下原则：

  • 不留痕迹：插件不得在笔记中留下任何特定于插件的配置，例如自定义前置属性。笔记可能会与不使用 Obsidian 的同事和团队共享。如果用户停止使用此插件，他们不应该清理所有笔记。
  • 保持原生 ：插件的外观和感觉应该像是 Obsidian 原生的。插件还应尽可能优先使用原生 Web API 而不是自定义组件。
  • 稳定性高于功能：此插件面向对可靠性要求较高的专业人士。任何错误报告和可用性问题都将优先于新功能。

Obsidian Theme - Minimal

发表于2024-04-29|更新于2024-06-09|ObsidianTheme
|字数总计:1.9k|阅读时长:6分钟|阅读量:

简介

在 minimal.guide 找到详细文档。

Minimal 是一款适用于桌面、手机和平板电脑的 Obsidian 主题。使用配套插件 Minimal Theme Settings 和 Hider 可自定义颜色、字体等。荣获 Obsidian 官方最佳主题奖 ✨ 。

支持开发： 关注我 或 请我喝咖啡 。

屏幕截图

背景样式包括低对比度、高对比度和真黑色选项，是 OLED 设备的理想选择。

用于表格和卡片布局的辅助类可让您创建强大的工作流程。

将聚焦模式与图像网格和图像宽度选项结合起来，您就可以将聚焦模式与图像网格和图像宽度选项结合起来。

Minimal 包含对数十种流行插件的支持，如 Calendar、Kanban、Dataview、Outliner、Excalidraw 以及更多 。

安装

安装主题

1.打开 Obsidian Setting ；
2.进入 Appearance 并点击 Manage ；
3.在社区主题下搜索 Minimal 并点击 Use 。

安装配套插件

1.转到 Community plugins 并关闭 Restricted mode ；
2.在社区插件下搜索 Minimal Theme Settings 并点击 Install ，然后点击 Enable 。

有问题？加入官方Obsidian Discord上的Minimal channel。

配套插件

Minimal Theme Settings plugin 允许您自定义配色方案、字体、热键，以及访问主要功能切换。强烈推荐所有 Minimal 用户使用该插件。

建议使用 Hider plugin 隐藏 Obsidian UI 元素，如窗口边框、滚动条、工具提示等。使用 Hider 关闭主工具栏（又称应用功能区）时，可将鼠标悬停在窗口左下角边缘。

Style Settings plugin 可让您创建自定义配色方案。此插件为可选项。

设置

可使用 Minimal Theme Settings 插件访问以下设置。下面的许多设置都可以通过 热键进行切换。

界面颜色和字体

  • Background contrast - 包括 OLED 设备的真黑暗色模式。
  • Color scheme - 包括 Catppuccin、Dracula、Everforest、Gruvbox、macOS、Nord、Solarized 以及更多 。
  • Line width - 设置窗格内的正常线宽和最大线宽。

你还可以使用 Style Settings plugin 插件进行更深入的定制。

功能

  • Text labels for primary navigation - 为左上角的导航添加文本标签，注意英语以外的语言需要翻译片段关于本地化的更多信息 。
  • Focus mode - 隐藏标题栏和状态栏。
  • Colorful frame - 在应用程序窗口顶部区域使用强调色。
  • Colorful headings - 为每个标题使用不同的颜色。
  • 隐藏工作区边框 - 去除分隔线，使外观更简约。
  • 修剪文件名 - 将文件名修剪为一行。

布局选项

控制图像、表格和 iframe。这些设置也可以使用 helper classes按文件启用。

  • Image grids - 将连续的图像变成列。要创建新行，请在图片之间添加额外的换行符。
  • Block widths - 按类型控制元素的宽度。

配色方案

预设配色方案

Minimal 提供配色方案，可使用 Minimal Theme Settings 插件分别选择明暗模式。配色方案包括 Dracula、Everforest、Gruvbox、macOS、Nord、Notion、Solarized 和 Things。

自定义配色方案

你可以使用 Style Settings plugin 插件自定义配色方案。要更方便地查看更改，请使用命令调板（默认为 CMD + P）在新窗格中打开 Style Settings 。使用 Style Settings ，您可以自定义 Minimal 的颜色、字体大小、字体样式以及更多细节。

插件支持

大多数插件都能很好地与 Minimal 配合使用，但以下插件受到了特别的关爱和关注：

  • 日历 Calendar
  • 图表 Charts
  • 检查事项 Checklist
  • 数据视图 Dataview
  • 字典 Dictionary
  • 绘图 Excalidraw
  • Git
  • 隐藏 Hider
  • 看板 Kanban
  • 地图传单 Leaflet for maps
  • 大纲视图 Outliner
  • 快速添加 QuickAdd

辅助过滤器和类

图像过滤器

添加图像滤镜可产生以下效果。使用方法是在图像链接的末尾添加过滤器名称，例如![[image.jpeg#invert]]。

过滤器 说明
#invert 在暗色模式下反转图像——适用于浅色背景上的图表和手写体。
#invertW 在浅色模式下反转图像——适用于深色背景上的图表和手写体。
#circle 将图像裁剪成圆形
#outline 在图像周围添加轮廓

表格、图像和 iframe 的线宽控制

按类型控制元素的宽度。了解更多。

类 说明
table-100, img-100, iframe-100 填充窗格宽度的 100%
table-max, img-max, iframe-max 填充最大线宽（默认88%）
table-wide, img-wide, iframe-wide 填充宽线宽度

卡片

使用 cards 可将 Dataview 表格转换为卡片。与表格宽度类兼容。 请注意，必须有 cards 才能使用其他类。了解更多。)

类 说明
cards (必填) 将所有数据视图表设置为卡片布局
cards-align-bottom 将卡片的最后一个元素对齐到底部
cards-cover 调整图像大小以填充定义的空间
cards-16-9 将卡片中的图片调整为 16:9 的比例
cards-1-1 将卡片中的图片调整为 1:1 的比例（正方形）
cards-2-1 以 2:1 的比例调整卡片中的图像
cards-2-3 按 2:3 的比例调整卡片中的图像
cards-cols-1 to 8 强制指定列数（从 1 到 8）

表格和数据视图

控制表格的行和列样式。了解更多。

类 说明
table-nowrap 在表格单元格中禁用换行
table-wrap 强制表格单元格换行
table-numbers 为表格添加行号
table-tabular 在表格中使用表格数字
table-small 在表格中使用小号字体
table-tiny 在表格中使用小号字体
table-lines 在所有表格单元格周围添加边框
row-lines 在表格行间添加边框
col-lines 在表格列之间添加边框
row-alt 为交替的表格行添加条纹背景
col-alt 为交替的表格列添加条纹背景

嵌入和反向夹杂

类 说明
embed-strict 在文本流中无缝显示转包内容。

- [/] incomplete 未完成
- [x] done 已完成
- [-] canceled 取消
- [>] forwarded 已转发
- [<] scheduling 计划安排
- [?] question 问题
- [!] important 重要
- [*] star 星级

- [l] location 位置
- [b] bookmark 书签
- [i] information 信息
- [S] savings 储蓄

- [c] cons 缺点

- [k] key 关键
